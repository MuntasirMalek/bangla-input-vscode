# Changelog

All notable changes to "Bangla Input" extension will be documented in this file.

## [2.4.0] - 2024-12-23

### Added
- Google API integration for accurate transliteration
- 5 suggestions popup via Cmd+G
- Auto-convert on space with first Google suggestion
- Offline fallback with 200+ word dictionary

### Changed
- Simplified keyboard shortcuts (Option+G toggle, Cmd+G suggestions)

## [1.0.0] - 2024-12-23

### Added
- Initial release
- Basic Banglish to Bengali transliteration
- Phonetic rules based on Avro patterns
- Toggle on/off functionality
- Status bar indicator
