# Changelog

All notable changes to "Bangla Input" extension will be documented in this file.

## [1.1.0] - 2025-01-02

### Added
- Bengali number conversion (16 → ১৬)
- Number key selection (1-5) in suggestion popup

### Changed
- Keyboard shortcuts changed to Option+B and Option+Shift+B

## [1.0.0] - 2025-01-01

### Added
- Google API integration for accurate transliteration
- 5 suggestions popup via Option+B
- Auto-convert mode (toggle with Option+Shift+B)
- Offline fallback with 600+ word dictionary
- Dari support (. → ।)
- Status bar indicator
